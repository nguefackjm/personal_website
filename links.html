Linux Journey

Linux Journey is a great resource available free of charge for learning to use Linux. As they put it, you will "learn the ways of Linux-fu, for free". It will guide the traveler on their journey from beginner to expert. It covers in a brief and easy to understand manner issues such as history of Linux, various distributions, command line usage, text manipulation, permissions, devices, file systems and a lot more.

I have been a mac OSX user for the last 7 years and just three months ago decided to start learning to use Linux systems. I installed a copy each of Kali Linux and Ubuntu on a virtual box running on my mac OSX. My experience with both the operating systems and the learning guide has been nothing but positive. In subsequent posts, I will illustrate the strengths of kali linux in networking and make a simple guide on how to install it in virtualbox and guest additions. Many thanks to the owners of the site and open software community.

Github Education

I would normally not feature github on this because it is very popular and I am just getting to know how to use it. I use this very easy to follow video. I featured the education version because this site was made possible by resources provided for free by github.

With a ".edu" email account on git education, you get free or discounted access to services such as AWS educate(this site is hosted on there), namecheap(where the site's domain name was obtained for free), Microsoft Azure and many more. A great thanks to Github Education and all participating companies. Also, many thanks to my good friend who not only referred me to github educate, but also bombards me with endless streams of many other great resources.

HTML Resources

After a long while without doing any front end design, using the wto sites: w3schools.com and General Assembly, I was able to keep up technically with the demands of making this site in no time. A great thanks to them and I recommend them for any novice html developer.

imgur

imgur is a great image hosting site. It loads real fast and provides you with tools to edit your images with an easy to use interface. I recommend it for projects where the user's hosting services is limited by space or the user is ready to deal with image editing tools like photoshop. As one of those users my self, all images her are hosted on imgur.
